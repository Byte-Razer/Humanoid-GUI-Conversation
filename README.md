Our School has a Humanoid Project.
This Repository contains the code made for the Humanoid Bot.
Humanoid.py contains the main code including a futuristic and alien like GUI
recognise.py contains the code for people recognition and is run when Start Recognition is clicked.
queries.py contains the code for the user to interact with the bot and is run when the Listen button is clicked
train.py contains the code for registering new users by cling Train Data button
